# Descripción para CV - Proyecto Stock

## Versión Corta (1 línea)

Desarrollo de sistema de gestión de inventario full-stack con Flask, React y React Native, incluyendo autenticación JWT, testing con pytest, arquitectura modular y documentación API con Swagger.

## Versión Media (2-3 líneas)

Desarrollo completo de sistema de gestión de inventario full-stack utilizando Flask (Python) para backend REST API y React para frontend web. Implementación de autenticación JWT, rate limiting, logging estructurado, y testing automatizado con pytest (cobertura >80%). Incluye aplicación móvil React Native, documentación API con Swagger, y arquitectura modular siguiendo Application Factory Pattern.

## Versión Completa (Para LinkedIn/Portfolio)

**Sistema de Gestión de Inventario Full-Stack**

Desarrollo de sistema completo de gestión de inventario para pequeñas y medianas empresas, abarcando backend, frontend web y aplicación móvil.

**Backend (Flask/Python):**
- Arquitectura modular con Application Factory Pattern
- API REST con autenticación JWT y renovación automática
- Configuración por entornos (development/testing/production)
- Rate limiting para protección contra abuso
- Logging estructurado con JSON
- Validación robusta de entrada y manejo centralizado de errores
- Testing completo con pytest (cobertura >80%)
- Documentación API interactiva con Swagger/Flasgger
- Soporte para SQLite (desarrollo) y PostgreSQL (producción)

**Frontend (React):**
- Patrón Flux para gestión de estado
- Componentes modulares y reutilizables
- Escaneo de códigos de barras (cámara e imagen) con Quagga2
- Búsqueda avanzada con filtros múltiples y paginación
- Tests con React Testing Library
- ESLint y Prettier para calidad de código

**Mobile (React Native):**
- Aplicación nativa para iOS y Android
- Autenticación JWT
- Consulta de inventario en tiempo real
- Búsqueda optimizada

**DevOps y Calidad:**
- Docker y Docker Compose para despliegue
- CI/CD con GitHub Actions (tests automáticos, auditoría de seguridad)
- Dependabot para actualizaciones automáticas
- Análisis y corrección de vulnerabilidades de seguridad

**Tecnologías:** Flask, SQLAlchemy, React, React Native, JWT, PostgreSQL, Docker, pytest, Swagger, GitHub Actions
